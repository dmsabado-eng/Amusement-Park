# Lab 2 - Amusement Park Ride Checker

## Summary
For this lab, I created an interactive C++ program for an amusement park. The program asks the user for their age and height, then decides if they are eligible to go on a ride.

## How I Did It
I used `cin` to collect the user's age and height. I used `cout` to display the prompts and results. Then I used `if`, `else if`, and `else` statements to check if the user meets the ride requirements.

The user must be at least 12 years old and at least 48 inches tall to ride.
